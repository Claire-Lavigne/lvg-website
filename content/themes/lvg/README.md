# LVG STUDIO

## Client
Mr Laurent Lavigne, photographe professionnel, sous l'entreprise LVG Studio, a créé son site web sur Wix.
Il souhaite moderniser son site, dans le but d'attirer davantage de clients et de fidéliser la clientèle. Tout en gardant le bon référencemment qu'il a actuellement et une partie du contenu déjà en ligne.
